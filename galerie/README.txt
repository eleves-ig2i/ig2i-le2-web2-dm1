INSTALLATION DU SITE
---------------------------
- Créer la base de données 'galerie' sur phpmyadmin, et appliquer le script SQL galerie.sql sur le site.
- Modifier le fichier config.php, de façon à ce que le site puisse intéragir avec la base de données 'galerie'.
- Sur le navigateur WEB, aller sur la page explorateur.php (ne pas double cliquer sur le fichier).
